# ♛ رُقي — للمطابخ والديكورات وغرف النوم

موقع إلكتروني احترافي لشركة **رُقي** المتخصصة في تصميم وتنفيذ المطابخ والديكورات وغرف النوم في طرابلس، ليبيا.

## 🌐 الموقع المباشر

بعد تفعيل GitHub Pages سيكون الرابط:  
`https://YOUR-USERNAME.github.io/ruqi-website/`

## 📁 محتوى المشروع

```
ruqi-website/
├── index.html        ← الموقع الكامل
└── README.md         ← هذا الملف
```

## 🚀 رفع الموقع خطوة بخطوة

### 1. إنشاء المستودع على GitHub
- اذهب إلى github.com وسجّل دخولك
- اضغط **New repository**
- اسم المستودع: `ruqi-website`
- اختر **Public**
- اضغط **Create repository**

### 2. رفع الملفات
- اضغط **Add file** ← **Upload files**
- ارفع `index.html` و `README.md`
- اضغط **Commit changes**

### 3. تفعيل GitHub Pages
- اذهب إلى **Settings** ← **Pages**
- في قسم **Source** اختر: `Deploy from a branch`
- Branch: `main` / Folder: `/ (root)`
- اضغط **Save**
- انتظر دقيقة وسيظهر رابطك الرسمي ✅
